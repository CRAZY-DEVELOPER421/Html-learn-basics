üì± LEVEL 1 - BEGINNERS

1. Viewport Meta Tag

<meta name="viewport" content="width=device-width, initial-scale=1.0">

Yeh tag HTML ke <head> section me dalna zaroori hai!

Kya hai: Mobile devices ko batata hai ki page ko kaise display kare.
Kab use kare: Har responsive website me - without this mobile pe zoom out dikhega.
Kaise kaam karta hai: width=device-width (device ki width use karo), initial-scale=1.0 (zoom level 100%).

2. Basic Media Queries - max-width

Desktop View (Blue)
@media (max-width: 768px) { background: red; }
@media (max-width: 480px) { background: green; }
Kya hai: max-width matlab "jab screen is size se CHOTI ho".
Kab use kare: Desktop-first approach me - bade se chhote ki taraf.
Kaise kaam karta hai: Browser window resize karo aur color change dekho! 768px se neeche red, 480px se neeche green.

3. Basic Media Queries - min-width (Mobile-First) ‚≠ê

Mobile View (Green)
/* Default: Mobile */
@media (min-width: 481px) { /* Tablet */ }
@media (min-width: 769px) { /* Desktop */ }
Kya hai: min-width matlab "jab screen is size se BADI ho" - Mobile-first approach!
Kab use kare: Modern approach - pehle mobile design, phir bade screens.
Kaise kaam karta hai: Default green (mobile), 481px+ red (tablet), 769px+ blue (desktop). Resize karke dekho!

Kya hai: Industry standard screen sizes jahan layout change karte hain.
Kab use kare: Starting point ke liye - sabse common devices cover ho jayenge.
Kaise kaam karta hai: In widths pe design test karo. Apne content ke base pe adjust kar sakte ho.

5. Fluid Layouts - Percentage Width

Width: 100% (Full width)
Width: 50% (Half width)
Width: 33.33% (One-third)
.box { width: 100%; } ‚Üê Mobile
@media (min-width: 768px) { .box { width: 50%; } } ‚Üê Tablet
@media (min-width: 1024px) { .box { width: 33.33%; } } ‚Üê Desktop
Kya hai: Fixed pixels ki jagah percentage use karna - parent ke according adjust hoga.
Kab use kare: Flexible layouts ke liye - screen size change hone pe automatically adjust.
Kaise kaam karta hai: 50% = parent ki half width. Browser resize karo, width adjust hoti hai!

6. Container with max-width

Container: max-width: 600px, width: 100%
Window choti karo - 600px se kam me 100% width lega, zyada me 600px max!
.container {
  max-width: 1200px;
  width: 100%;
  margin: 0 auto;
  padding: 0 20px;
}
Kya hai: Container ki maximum width set karna - bade screens pe content bahut wide na ho.
Kab use kare: Main content wrapper ke liye - readability improve hoti hai.
Kaise kaam karta hai: max-width: 1200px (bada se zyada nahi), width: 100% (choti screens pe full), margin: 0 auto (center align).

7. Responsive Images

Window resize karo - image shrink hoti hai but aspect ratio same rahta hai!

img {
  max-width: 100%;
  height: auto;
  display: block;
}
Kya hai: Images ko responsive banana - container se badi na ho.
Kab use kare: Har image ke liye - mobile pe overflow nahi hogi.
Kaise kaam karta hai: max-width: 100% (parent se bada nahi), height: auto (proportions maintain), display: block (extra space remove).

9. CSS Units - rem, em, vw, vh

16px (Fixed) - hamesha 16px
1rem (Root relative) - root font-size ke equal
2em (Parent relative) - parent ke double
3vw (Viewport width) - screen width ka 3%
20vh (Viewport height) - screen height ka 20%
font-size: 16px; ‚Üê Fixed
font-size: 1rem; ‚Üê Root relative (usually 16px)
font-size: 2em; ‚Üê Parent relative
font-size: 3vw; ‚Üê 3% of viewport width
height: 20vh; ‚Üê 20% of viewport height
Kya hai: Different measurement units - fixed aur relative.
Kab use kare: rem (consistent sizing), em (component-based), vw/vh (full-screen sections).
Kaise kaam karta hai: px = fixed, rem = root based (1rem = 16px default), em = parent based, vw/vh = screen based. Window resize karo aur vw/vh ka effect dekho!

12. Responsive Typography - clamp()

Responsive Heading

Yeh text automatically resize hota hai! Window ko chhota-bada karo aur dekho font size smoothly adjust hota hai.

h2 {
  font-size: clamp(1.5rem, 5vw, 3rem);
}
p {
  font-size: clamp(1rem, 2.5vw, 1.5rem);
}
Kya hai: clamp(minimum, preferred, maximum) - fluid typography without media queries!
Kab use kare: Headings aur body text - smooth scaling chahiye.
Kaise kaam karta hai: clamp(1.5rem, 5vw, 3rem) = minimum 1.5rem, prefer 5% viewport, max 3rem. Automatically scale hota hai!
