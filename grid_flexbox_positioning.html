7. Display, Visibility & Overflow

display: block

Block Element 1
Block Element 2
Kya hai: Element puri width le leta hai aur naye line pe start hota hai.
Kab use kare: Div, headings, sections ke liye.
Kaise kaam karta hai: Har element neeche wali line pe chala jata hai, puri horizontal space occupy karta hai.

display: inline

Inline 1 Inline 2 Inline 3
Kya hai: Element sirf apne content jitna width leta hai, same line pe rehta hai.
Kab use kare: Text ke beech elements ke liye (span, a, strong).
Kaise kaam karta hai: Width/height set nahi kar sakte, text flow ke saath chalta hai.

display: inline-block

Box 1
Box 2
Box 3
Kya hai: Inline ki tarah same line pe, but width/height set kar sakte ho.
Kab use kare: Buttons, cards, navigation items ke liye.
Kaise kaam karta hai: Best of both - inline flow + block properties.

display: none

Visible Element
Hidden Element (display: none)
Visible Element (koi gap nahi hai)
Kya hai: Element completely hide, space bhi nahi leta.
Kab use kare: Toggle functionality, conditional rendering.
Kaise kaam karta hai: Element DOM me hai but render nahi hota, space occupy nahi karta.

visibility: hidden

Visible
Hidden (visibility: hidden)
Visible (gap hai upar)
Kya hai: Element invisible but space occupy karta hai.
Kab use kare: Jab layout disturb na ho but element hide karna ho.
Kaise kaam karta hai: Space leta hai but dikhta nahi.

opacity

Opacity: 1 (100% visible)
Opacity: 0.5 (50% transparent)
Opacity: 0.2 (20% visible)
Kya hai: Element kitna transparent hoga (0 to 1).
Kab use kare: Fade effects, hover states, overlays.
Kaise kaam karta hai: 0 = fully transparent, 1 = fully visible.

overflow: hidden

Yeh bahut lamba text hai jo box se bahar ja raha hai. Overflow hidden hai to extra content cut ho jayega aur scrollbar bhi nahi aayega. Lorem ipsum dolor sit amet.
Kya hai: Extra content ko hide kar do.
Kab use kare: Clean look chahiye, content limit karna ho.
Kaise kaam karta hai: Jo content box se bahar hai wo cut ho jata hai.

overflow: scroll

Yeh bahut lamba text hai. Scrollbar hamesha rahega chahe content chota ho ya bada. Tum scroll karke pura content padh sakte ho. Lorem ipsum dolor sit amet consectetur adipisicing.
Kya hai: Hamesha scrollbar show karo.
Kab use kare: Jab tum chahte ho user scroll karke pura content dekhe.
Kaise kaam karta hai: Scrollbar always present (content chota ho tab bhi).

overflow: auto

Content overflow hoga tabhi scrollbar aayega. Auto smartly decide karta hai. Yeh practical approach hai. Lorem ipsum dolor sit amet consectetur.
Kya hai: Zarurat hone pe scrollbar show karo.
Kab use kare: Most common use case - smart behavior.
Kaise kaam karta hai: Content overflow = scrollbar, warna nahi.

8. Positioning & Layering

position: static

Static Element (default position)
Kya hai: Default position, normal document flow me rehta hai.
Kab use kare: Default hai, explicitly likhne ki zarurat nahi.
Kaise kaam karta hai: top, left properties kaam nahi karti.

position: relative

Relative (top: 20px, left: 30px)
Kya hai: Apni original position se adjust kar sakte ho.
Kab use kare: Slight adjustments ke liye, absolute positioning ka parent.
Kaise kaam karta hai: Original space occupied rehta hai, element move hota hai. top, right, bottom, left use kar sakte ho.

position: absolute

Normal Content (gray background hai container)
Absolute (top: 50px, right: 20px)
Kya hai: Positioned parent ke relative me position hota hai.
Kab use kare: Tooltips, dropdowns, overlays ke liye.
Kaise kaam karta hai: Document flow se nikal jata hai, space occupy nahi karta. Nearest positioned parent (relative/absolute/fixed) se position hota hai.

position: fixed

Scroll karo page ko, neeche right corner me green box fixed rahega!
(Green box viewport ke saath fixed hai)
Fixed Box
Kya hai: Viewport (screen) ke relative me fixed rehta hai.
Kab use kare: Sticky headers, floating buttons, chat widgets.
Kaise kaam karta hai: Scroll karne pe bhi same jagah pe rahta hai.

position: sticky

Content upar
Sticky Header (scroll karo)
Bahut saara content neeche... scroll karo aur dekho sticky header upar stick ho jata hai!
Kya hai: Scroll karne pe ek position pe stick ho jata hai.
Kab use kare: Table headers, section headings.
Kaise kaam karta hai: Relative ki tarah behave karta hai jab tak threshold cross na ho, phir fixed ban jata hai.

top, right, bottom, left

top: 10px, left: 10px
bottom: 10px, right: 10px
Kya hai: Positioned elements ko move karte hain.
Kab use kare: position: relative/absolute/fixed/sticky ke saath.
Kaise kaam karta hai: top: element ko neeche move, left: element ko right move karta hai (aur vice versa).

z-index

z-index: 1
z-index: 2
z-index: 3
Kya hai: Overlapping elements ka stacking order control karta hai.
Kab use kare: Modals, dropdowns, layered designs.
Kaise kaam karta hai: Higher z-index = upar layer me. Sirf positioned elements pe kaam karta hai.
